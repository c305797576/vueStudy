## 热门图书的功能
- 先写服务端，确保数据能正常返回
- 增加api方法，实现调取数据的功能
- 在哪个组建者应用这个api，如果是一个基础组件需要用这些数据，在使用这个组件的
父级中调用这个方法，将数据传递给组件
- 写一个基础组件
  - 1.创建.vue文件
  - 2.在需要使用这个组件的父级中引用这个组件
  - 3.在组建者注册
  - 4.以标签的形式引入
